Colors in Flutter

  1. Color(int value)

    2. Construct a color from the lower 32 bits of an int.
    A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. All values must be between 00 and FF.

  2. Color.fromARGB(int a, int r, int g, int b)

    3. Construct a color from the lower 8 bits of four integers:
    - a is the alpha value, with 0 being transparent and 255 being fully opaque.
    - r is red, from 0 to 255.
    - g is green, from 0 to 255.
    - b is blue, from 0 to 255.

  3. Color.fromRGBO(int r, int g, int b, double opacity)

    4. Create a color from red, green, blue, and opacity, similar to rgba() in CSS.
    - r is red, from 0 to 255.
    - g is green, from 0 to 255.
    - b is blue, from 0 to 255.
    - opacity is alpha channel of this color as a double, with 0.0 being transparent and 1.0 being fully opaque.

  4. Colors

    5. Defines the colors that most of the Material components use by default.

    Example

            Column(

              children: [

                Container(

                  height: 100,

                  color: Color(0xAF42A5F5),

                ),

                Container(

                  height: 100,

                  color: Color.fromARGB(255, 255, 165, 245),

                ),

                Container(

                  height: 100,

                  color: Color.fromRGBO(66, 32, 145, 1.0),

                ),

                Container(

                  height: 100,

                  color: Colors.purple,

                ),

              ],

            )




  5. Gradients

    6. Color gradients take a starting color and position and ending color and position. Then it performs a transition between the colors.

    Example :

            Container(

              decoration: BoxDecoration(

                  gradient: LinearGradient(

                    begin: Alignment.topRight,

                    end: Alignment.bottomLeft,

                    colors: [

                      Colors.blue,

                      Colors.red,

                    ],

                 )

              ),

            )
