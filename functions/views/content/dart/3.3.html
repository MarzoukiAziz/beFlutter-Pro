Set

  1. Definition

    2. Set is a unordered list of objects in which each object can occur only once.
    Sets are typed: if you specify the type of the set or Dart infers it then all the elements of the set must have the same type.


  2. Syntax

    3. var setName = <type>{}

    OR

    Set <type> setName={}


  3. Example

    4. var champs={"lux", "Diana", "Ahri"}


  4. Inserting Elements

    5. add(): insert single item into the set.
    addAll(): insert multiple items into the set.


    void main() {

      var  set = <int>{};

      print(set);

      set.add(1);

      print(set);

      set.addAll({2, 3, 4});

      print(set);

      set.add(1);

      set.addAll({3, 4});

      print(set);

    }

    {1}
    {1, 2, 3, 4}
    {1, 2, 3, 4}

  5. Removing Elements

    6. remove(value): Removes value from the set. Returns true if value was in the set. Returns false otherwise. The method has no effect if value value was not in the set.
    removeAll(elements): R emoves each element of elements from this set.
    clear(): Removes all elements in the set.


    void main() {

      var  set = <int>{0, 1, 2, 3, 4};

      print(set);

      set.remove(4);

      print(set);

      set.removeAll({2, 3});

      print(set);

      set.clear();

      print(set);

    }



    {0, 1, 2, 3, 4}
    {0, 1, 2, 3}
    {0, 1}
    {}

  6. Methods And Properties

    7. length: Returns the number of elements.
    contains(value): returns true if value is in the set.
    elementAt(index): returns the element at specified index postion(0 <=index <=size-1) .
    union(Set other): Returns a new set which contains all the elements of this set and other.
    intersection(Set other): Returns a new set which is the intersection between this set and other.
    difference(Set other): Returns a new set with the elements of this that are not in other.
    toList(): Creates a List containing the elements of this Set.


    void main() {

      var set = {1, 2, 3, 4, 5, 6, 7, 8};

      var set1 = {1, 3, 5, 7};

      var set2 = {2, 4, 6, 8};

      var set3 = {1, 3, 6};

      print(set.length);

      print(set1.contains(3));

      print(set2.contains(5));

      print(set.elementAt(3));

      var setUnion = set1.union(set2);

      print(setUnion);

      var setIntersection = set1.intersection(set2);

      print(setIntersection);

      var setDifference = set.difference(set3);

      print(setDifference);

      var listFromSet = [];

      listFromSet = set.toList();

      print(listFromSet);

    }
