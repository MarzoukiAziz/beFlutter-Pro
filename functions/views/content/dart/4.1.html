Assert

  1. Definition

    2. During development, use an assert statement to disrupt normal execution if a boolean condition is false.


  2. Syntax

    3. assert(<expppression>)

    OR

    assert(<expppression>,<Optional Message>)


  3. Example

    4. void main() {

      var nb = 6;

      assert(nb > 10, "You enter a number less than 10");

      print("Good Answer");

    }
