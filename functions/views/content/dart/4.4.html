While and do-while

  1. While

    2. The while statement continually executes a block of statements while a particular condition is true.

    1. Syntax

      2. while ( expression ) {
         statement(s)
      }


      The while statement evaluates expression, which must return a boolean value. If the expression evaluates to true, the while statement executes the statement(s) in the while block. The while statement continues testing the expression and executing its block until the expression evaluates to false.


    2. Example

      3. void main() {

        int count = 0;

        while (count < 5) {

          print('count = $count');

          count++;

        }

      }



      count = 0
      count = 1
      count = 2
      count = 3
      count = 4

    3. Infinite while loop

      4. void main() {

        int count = 10;

        while (count > 5) {

          print("count = $count");

          count++;

        }

      }



      This loop would never end, its an infinite while loop. This is because condition is count>5 which would always be true as we are incrementing the value of count inside while loop.
      Here is another example:


      void main() {

        while (true) {

          print("hello");

        }

      }



  2. Do-While

    3. The Dart programming language also provides a do-while statement.

    1. Syntax:

      2. do {
         statement(s)
      } while ( expression ) ;


      The difference between do-while and while is that do-while evaluates its expression at the bottom of the loop instead of the top. Therefore, the statements within the do block are always executed at least once.


    2. Example:

      3. void main() {

        var count = 3;

        do {

          print("count = $count");

          count++;

        } while (count < 7);

      }



      count = 3
      count = 4
      count = 5
      count = 6

      void main() {

        var count = 3;

        do {

          print("count = $count");

          count++;

        } while (count < 1);

      }


      count = 3

      We can use it in user input control:

      import 'dart:io';

       

      void main() {

        String input;

        bool isNumeric;

        int number;

        do {

          print("enter a number between 1 and 20");

          input = stdin.readLineSync();

          isNumeric = int.tryParse(input) != null;

          if (isNumeric == false)

            print("This is not a number");

          else

            number = int.tryParse(input);

        } while (!(isNumeric && number <= 20 && number >= 1));

        print("Good Job");

      }
