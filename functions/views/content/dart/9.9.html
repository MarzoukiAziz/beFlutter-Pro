Encapsulation

  1. What is encapsulation?

    2. The whole idea behind encapsulation is to hide the implementation details from users.

    If a data member is private it means it can only be accessed within the same class. No outside class can access private data member of other class.

    However if we setup public getter and setter methods to update the private data fields then the outside class can access those private data fields via public methods.

    This way data can only be accessed by public methods thus making the private fields and their implementation hidden for outside classes.

    Thatâ€™s why encapsulation is known as data hiding.


  2. Private Fields & Private methods

    3. If you want to restrict access to a certain field or method outside of a class, you can make it private by prefixing the field name with an underscore.

    Note :

    In Dart, the class must be in a separated file to restrict access to its private members. See these two examples to understand better.

    Example 1 :

        The class and the main method are in the same file.


    class User {

      String email;

      String name;

      String _password;

      User(this.email, this.name, this._password);

      void displayEmail() {

        print(email);

      }

     

      void _displayPassword() {

        print(_password);

      }

    }

     

    void main() {

      var x = User("xxx@gmail.com", "drX", "iamX");

      print(x._password);

      x._displayPassword();

    }

  3. Setters and Getters

    4. If we want to have more control over the field, we can use the "private field, public getter/setter" pattern.


    Getter : a method that use the get keyword with no parameter and a valid return type.

    Setter : a method that use the set keyword with one parameter and without return type.


    Example :

    user.dart

    class User {

      String email;

      String name;

      String _password;

      User(this.email, this.name, this._password);

     

      String getPassword() {

        return _password;

      }

     

      void setPassword(String x) {

        _password = x;

      }

    }



    main.dart


    import 'user.dart';

     

    void main() {

      var x = User("xxx@gmail.com", "drX", "iamX");

      print(x.getPassword());

      x.setPassword("iamnotXanymore");

      print(x.getPassword());

    }
